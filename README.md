# ✈ Flight Radar

Real-time ADS-B flight tracker built with FastAPI and Leaflet.js.

## Live Demo
[flight-radar-mb39vr1or-hichemuuus-projects.vercel.app](https://flight-radar-iota.vercel.app/)

## Features
- 9,000+ live aircraft from OpenSky Network
- Cluster map with altitude color coding
- Click any aircraft for detailed info
- Filter by country
- Real-time stats

## Stack
- **Backend**: Python, FastAPI, httpx
- **Frontend**: HTML, CSS, Vanilla JS, Leaflet.js
- **Data**: OpenSky Network ADS-B API
- **Deploy**: Railway + Vercel
